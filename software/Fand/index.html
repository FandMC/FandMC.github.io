Minecraft 服务端

Fand 是一个基于 Leaves 的 Minecraft 游戏服务端，旨在修复原版特性的缺陷。

为什么选择 Fand?

速度极快

Fand 基于Leaves包含许多改进和优化,显著提高了性能。这包括异步区块加载,以及对光照引擎、漏斗、实体等的重大优化。

扩展的 API

Fand 扩展和改进了 Bukkit、Spigot、Leaves 和 Paper 的 API ,使您和您的开发者拥有更多的功能和特性。

快速开始

要开始使用 Fand，您需要下载并安装最新版本的服务器软件。一旦准备就绪，请查看我们详尽的文档。
